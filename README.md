# Juan Galarce | Portfolio 2026

Portafolio profesional de Juan Galarce, Desarrollador Full Stack Junior especializado en React, Inteligencia Artificial y tecnologías web modernas.

## Descripción

Este proyecto es una aplicación web single-page (SPA) construida con tecnologías modernas para mostrar experiencia, proyectos y habilidades técnicas.

**Stack Tecnológico:**
- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Vite](https://vitejs.dev/)

## Ejecutar Localmente

1. Instalar dependencias:
   ```bash
   npm install
   ```

2. Iniciar el servidor de desarrollo:
   ```bash
   npm run dev
   ```

3. Construir para producción:
   ```bash
   npm run build
   ```

## Despliegue

La aplicación incluye un archivo `vercel.json` configurado para asegurar el correcto enrutamiento SPA en Vercel.
